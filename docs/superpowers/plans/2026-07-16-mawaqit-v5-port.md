# Mawaqit v4→v5 Port Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port the Noctalia v4 (QML) "mawaqit" prayer-times plugin to the v5 (Luau) plugin system as `aalbayaty/mawaqit`, with full feature parity minus azan audio, inside a reusable multi-plugin monorepo.

**Architecture:** `service.luau` is the sole owner of all logic (Aladhan fetch, countdown, Hijri/calendar math, events, hadith, notifications) and publishes display-ready values to `noctalia.state`; `widget.luau` and `panel.luau` are thin renderers that read state and send back command tables. There is no `require` at runtime (the sandbox forbids it), so nothing is shared across entries. Pure logic lives as `local`s inside `service.luau`, exposed for off-shell unit testing via an `__TEST` export that the `luau` CLI loads with `require`.

**Tech Stack:** Luau (`--!nonstrict`), Noctalia v5 plugin API (`noctalia.*`, `ui.*`, `barWidget.*`, `panel.*`), Aladhan HTTP API, TOML manifest. Tooling: `luau` (run tests), `luau-analyze` (typecheck/lint), `python3` (manifest validator + catalog generator).

## Global Constraints

These apply to every task; copied verbatim from the spec.

- Plugin identity: `id = "aalbayaty/mawaqit"`, `author = "aalbayaty"`, `min_noctalia = "5.0.0"`, one top-level `mawaqit/` dir in the repo.
- Full feature parity **minus azan audio**: no bundled mp3s, no `playAzan`/`azanFile` settings, no bar volume/stop UI, no azan preview. Notifications remain.
- **No `require` at runtime.** `service.luau` is the sole logic owner; `widget.luau`/`panel.luau` are thin state-renderers. Confirmed: mpvpaper source comments "Noctalia plugin scripts run in an isolated sandbox without `require`."
- Settings are **plugin-level `[[setting]]`** in `plugin.toml`, auto-generated UI; read with `noctalia.getConfig(key)`; there is **no `setConfig`** (durable data goes to `pluginDataDir()` files).
- Notifications are title+body only via `noctalia.notify` (drop v4's urgency/icon/`notify-send`).
- Add a `use12hourFormat` bool setting (default `false`) — v5 does not expose the shell's global 12h flag to plugins.
- Aladhan API base: `https://api.aladhan.com/v1`. No auth/keys. All GET, async via `noctalia.http`.
- Monorepo validator rule: `id` must equal `"<author>/<dirname>"`. Never hand-edit `catalog.toml` (regenerated by `update-catalog.py`).
- Entry-file guard (every entry ends with it): `if _G.__MAWAQIT_TEST or noctalia == nil then return M end` — production has `noctalia` set and the flag nil, so the guard is skipped and the chunk runs normally; the `luau` test harness `require`s the file before injecting globals, so `noctalia == nil` returns `M`.
- All commands run from repo root `/home/ahmad/Source/noctalia-plugins` (git repo, branch `main`, identity configured). `luau`, `luau-analyze` on PATH (`~/.local/bin`); `python3` is 3.13 with `tomllib`.
- Test commands: `luau mawaqit/tests/<file>.luau` (exit 0 = pass; `luau -` stdin is NOT supported). `luau-analyze mawaqit/<file>.luau` (exit 0 clean); entry files emit expected `Unknown global 'noctalia'/'ui'/'barWidget'/'panel'/...` lines that MUST be filtered with `grep -v "Unknown global '\(noctalia\|ui\|barWidget\|shortcut\|launcher\|desktopWidget\|panel\)'"` before asserting cleanliness. `python3 .github/workflows/validate-plugins.py` ("Validated N plugin manifest(s).").

### Locked interface contract (state keys, `service.luau` internals `M.*`, harness `H.*`)

State keys the service sets (widget/panel only read them; all display-ready):
`status` ("loading"|"ok"|"error"), `error` (string), `prayers` (`{{key,label,time,isNext,highlight}}`, highlight ∈ none|next|imsak|maghrib, order Imsak,Fajr,Sunrise,Dhuhr,Asr,Maghrib,Isha), `hijri` (`{display,displayAr,day,month,year,monthNameEn,monthNameAr,days}`), `gregorian` (string), `isRamadan` (bool), `isJumuah` (bool), `event` (`{whenText,nameAr}`|nil), `hadith` (`{text,source}`), `calendar` (`{month,year,monthNameEn,synced,syncedAt,gregRange,weekStartDay,leadingBlanks,cells:{{hday,gday,isToday,isPast,isFriday,eventAr}}}`), `countdown` (`{next,label,secondsToNext,text,mode,color,glyph,elapsed}`, mode ∈ loading|error|nodata|countdown|now|elapsed|time, color ∈ primary|secondary|tertiary), `command` (`{action,ts,month?,year?}`, action ∈ refresh|calendar|calendarRefresh; set by widget/panel, watched by service).

`service.luau` internal functions on `M` (pure unless noted): `buildTuneParam(cfg)`, `buildWeeklyUrl(cfg,ddmmyyyy)`, `buildSingleDayUrl(cfg)`, `cleanTime(raw)`, `normalizeWeekly(decoded,todayISO)`, `normalizeSingleDay(decoded)`, `computeCountdown(timings,nowSec,cfg,ctx)`, `buildPrayerRows(timings,cfg,ctx,nextKey)`, `toJDN(y,m,d)`, `jdnToGreg(jdn)`, `parseReadableToJDN(readable)`, `weekdayColumn(jdn,weekStartDay)`, `isHijriLeapYear(hyear)`, `estimateMonthLength(hmonth,hyear)`, `eventsFor(hmonth,hday)`, `upcomingEvent(hmonth,hday,gregWeekday)`, `hadithForDay(dayOfYear)`, `isLastTenNights(hmonth,hday,monthDays)`, `formatTime(hhmm,use12h)`, `toArabicNumerals(s)`, `formatCountdownText(seconds)`, `prayerGlyph(key,isJumuah)`, `colorRole(settingValue)`, `buildCalendar(month,year,hijriCtx,calData,cfg)`. Constants: `M.PRAYER_ORDER`, `M.PRAYER_KEYS`, `M.NOTIFY_KEYS`, `M.METHODS`, `M.MONTH_NAMES_EN`, `M.MONTH_NAMES_AR`, `M.EVENTS`, `M.HADITH`, `M.PRAYER_GLYPHS`.

Harness `H` (built in Task 1.3): `H.installGlobals()`, `H.ui.*` (each returns `{type,props,children}`), `H.state` (get/set/watch), `H.calls` (recorders: render, tooltip, togglePanel, command, notify, http), `H.loadEntry(relpath)` (require + inject globals into module env, sets `__MAWAQIT_TEST`), `H.env` (the loaded entry's env, for calling globals like `onClick`), `H.eq/H.ok/H.truthy`, `H.test(name,fn)`, `H.finish()`, `H.find(tree,pred)` (DFS tree search).

```lua
local ARABIC_DIGITS = { ["0"]="٠",["1"]="١",["2"]="٢",["3"]="٣",["4"]="٤",["5"]="٥",["6"]="٦",["7"]="٧",["8"]="٨",["9"]="٩" }

  return (tostring(s):gsub("%d", function(d) return ARABIC_DIGITS[d] end))
end

function M.formatTime(hhmm, use12h)
  local h, m = hhmm:match("^(%d+):(%d+)")
  if not h then return hhmm end
  h, m = tonumber(h), tonumber(m)
  if not use12h then return string.format("%02d:%02d", h, m) end
  local suffix = h < 12 and "AM" or "PM"
  local h12 = h % 12
  if h12 == 0 then h12 = 12 end
  return string.format("%d:%02d %s", h12, m, suffix)
end

function M.formatCountdownText(seconds)
  seconds = math.max(0, math.floor(seconds))
  local h = math.floor(seconds / 3600)
  local m = math.floor((seconds % 3600) / 60)
  local s = seconds % 60
  if h > 0 then return string.format("%dh %dm %ds", h, m, s) end
  if m > 0 then return string.format("%dm %ds", m, s) end
  return string.format("%ds", s)
end
```

```lua
local function ddmmyyyyToISO(d)
  local dd, mm, yy = d:match("^(%d%d)%-(%d%d)%-(%d%d%d%d)")
  if not dd then return nil end
  return string.format("%s-%s-%s", yy, mm, dd)
end

local function normalizeEntry(rawDate, rawTimings)
  local timings = {}
  for k, v in pairs(rawTimings) do timings[k] = M.cleanTime(v) end
  return {
    timings = timings,
    hijri = rawDate.hijri,
    gregorian = rawDate.gregorian and rawDate.gregorian.date or "",
    readable = rawDate.readable or "",
  }
end

function M.normalizeWeekly(decoded, todayISO)
  if type(decoded) ~= "table" or decoded.code ~= 200 or type(decoded.data) ~= "table" then return nil end
  local chosen
  for _, item in ipairs(decoded.data) do
    local iso = item.date and item.date.gregorian and ddmmyyyyToISO(item.date.gregorian.date)
    if iso == todayISO then chosen = item; break end
  end
  chosen = chosen or decoded.data[1]
  if not chosen then return nil end
  return normalizeEntry(chosen.date, chosen.timings)
end

function M.normalizeSingleDay(decoded)
  if type(decoded) ~= "table" or decoded.code ~= 200 or type(decoded.data) ~= "table" then return nil end
  return normalizeEntry(decoded.data.date, decoded.data.timings)
end
```

```lua
function M.toJDN(y, m, d)
  local a = math.floor((14 - m) / 12)
  local yy = y + 4800 - a
  local mm = m + 12 * a - 3
  return d + math.floor((153 * mm + 2) / 5) + 365 * yy + math.floor(yy / 4)
       - math.floor(yy / 100) + math.floor(yy / 400) - 32045
end

function M.jdnToGreg(jdn)
  local a = jdn + 32044
  local b = math.floor((4 * a + 3) / 146097)
  local c = a - math.floor((146097 * b) / 4)
  local d = math.floor((4 * c + 3) / 1461)
  local e = c - math.floor((1461 * d) / 4)
  local m = math.floor((5 * e + 2) / 153)
  local day = e - math.floor((153 * m + 2) / 5) + 1
  local month = m + 3 - 12 * math.floor(m / 10)
  local year = 100 * b + d - 4800 + math.floor(m / 10)
  return year, month, day
end

local MONTH_ABBR = { Jan=1, Feb=2, Mar=3, Apr=4, May=5, Jun=6, Jul=7, Aug=8, Sep=9, Oct=10, Nov=11, Dec=12 }
function M.parseReadableToJDN(readable)
  local d, mon, y = tostring(readable):match("(%d+)%s+(%a+)%s+(%d+)")
  if not d then return nil end
  local mm = MONTH_ABBR[mon]
  if not mm then return nil end
  return M.toJDN(tonumber(y), mm, tonumber(d))
end

function M.weekdayColumn(jdn, weekStartDay)
  local offset = (weekStartDay + 6) % 7
  return (jdn % 7 - offset + 7) % 7
end

local HIJRI_LEAP = { [2]=true,[5]=true,[7]=true,[10]=true,[13]=true,[16]=true,[18]=true,[21]=true,[24]=true,[26]=true,[29]=true }
function M.isHijriLeapYear(hyear)
  return HIJRI_LEAP[hyear % 30] == true
end
function M.estimateMonthLength(hmonth, hyear)
  if hmonth == 12 then return M.isHijriLeapYear(hyear) and 30 or 29 end
  return (hmonth % 2 ~= 0) and 30 or 29
end
```

```lua
function M._saveCache(cfg, entry)
  local blob = { entry = entry, city = cfg.city, country = cfg.country, method = cfg.method,
    school = cfg.school, fajrAngle = cfg.fajrAngle, ishaAngle = cfg.ishaAngle }
```

```lua
local M = {}

local function role(v) if v == nil or v == "" or v == "none" then return "on_surface" end return v end

function M.colorRoleFor(cd, cfg, which)
  if cd.mode == "now" or cd.mode == "elapsed" then return role(cfg.activeColor) end
  return role(which == "glyph" and cfg.iconColor or cfg.textColor)

function M.buildTree(cd, cfg, isVertical)
  local glyph = ui.glyph({ name = cd.glyph, color = M.colorRoleFor(cd, cfg, "glyph") })

  if isVertical then

  end
  return ui.row({ gap = 6, align = "center" }, { glyph, label })
```
